\section{Functional Equation Lemma}

In this section, we will discuss the formalization of functional equation lemma.
To begin with, we formalize the datum involved in the functional equation lemma as follow.
\begin{lstlisting}
variable {K : Type*} [CommRing K] {R : Subring K} {I : Ideal R}
  {p t q : ℕ} [hp : Fact (Nat.Prime p)] (ht : t ≠ 0) (hq : q = p ^ t)
  (σ : K →+* K) (s : ℕ → K) {g : PowerSeries R}
\end{lstlisting}
Let $K$ be a commutative ring, $R$ be a subring of $K$, $I$ be a
ideal of $R$. Let $p, q, t$ be natural number, where $p$ is a prime
number and $q$ can be written as $q = p ^ t$ with a nonzero $t$. Let $\sigma$ be a ring
endomorphism of $K$ and $s$ is a series of elements in $K$.

According to the equation $\ref{eq: coeff_RecurFun}$, we can inductively define a
function \lstinline{RecurFunAux} using equation $\ref{eq: coeff_RecurFun}$ and obtain a power series
using the API \lstinline{PowerSeries.mk}, then prove it satisfy
the functional equation $\ref{equation: recurFunEq}$, namely
\[f_g(X) = g(X) + \sum_{i=1}^{\infty} s_i \sigma_*^i f_g (X^{q ^ i}).\]
\begin{lstlisting}
def RecurFunAux (hg : g.constantCoeff = 0) : ℕ → K
  | 0 => 0
  | k + 1 =>
    g.coeff (k + 1) + ∑ j ∈ (Icc 1 (multiplicity q (k + 1))).attach,
      have : ((k + 1) / (q ^ (j : ℕ))) < k + 1 := ...
      (s j) * σ^[j] (RecurFunAux hg ((k + 1) / (q ^ (j : ℕ))))

def RecurFun := PowerSeries.mk (RecurFunAux ht hq σ s hg)
\end{lstlisting}

In order to inductively define \lstinline|RecurFunAux|, we prove that
for all \lstinline|j ∈ Icc 1 (multiplicity q (k + 1))|,
\lstinline|((k + 1) / (q ^ (j : ℕ))) < k + 1| in the motive of \lstinline|k + 1|. Therefore,
we need to use \lstinline|Finset.attach| to enable us to use the property of
\lstinline|j ∈ Icc 1 (multiplicity q (k + 1))|. To be precise,
let \lstinline|s| be a finite set, \lstinline{Finset.attach s} takes the elements of \lstinline|s|
and forms a new set of elements of the subtype \lstinline|{x // x ∈ s}|. With the property
\lstinline|j ∈ Icc 1 (multiplicity q (k + 1))|, we can prove that
\lstinline|((k + 1) / (q ^ (j : ℕ))) < k + 1|. Otherwise,
\lstinline|(RecurFunAux hg ((k + 1) / (q ^ (j : ℕ))))| is not valid.

To state the formalization of functional equation, we need APIs related to \lstinline|expand|,
which is a algebraic homomorphism that expand a
(multivariate) power series by a nonzero factor $p$. We will discuss these APIs first.

\begin{lstlisting}
variable {σ τ R S : Type*}  [CommRing R] [CommRing S] (p : ℕ) (hp : p ≠ 0)

/-- Expand the power series by a factor of p, so `∑ aₙ x^n` becomes `∑ aₙ x^(np)`. -/
noncomputable def expand : MvPowerSeries σ R →ₐ[R] MvPowerSeries σ R :=
  substAlgHom (HasSubst.X_pow hp)
\end{lstlisting}

We formalize functional equation $\ref{equation: recurFunEq}$ as follow.
\begin{lstlisting}
theorem FunEq_of_RecurFun [TopologicalSpace K] [T2Space K] (hs₀ : s 0 = 0) :
    let f := RecurFun ht hq σ s hg
    f = g.map R.subtype + ∑' (i : ℕ), s i • (f.expand (q ^ i) (q_pow_neZero hq)).map (σ^i) := ...
\end{lstlisting}

The additional instance \lstinline|[TopologicalSpace K]| provides a topological structure on
ring of power series over $K$ and \lstinline|[T2Space K]| ensure that the
infinite sum in the equation has unique limit, whenever it converges.
Let $R$ be a subring of $K$, then \lstinline|R.subtype|
is a canonical ring homomorphism from $R$ to $K$. \lstinline|PowerSeries.expand p hp f| expands a power
series \lstinline|f| by a factor of nonzero $p$.

The theorem \lstinline|coeff_one_RecurFun| is the 1-th coefficient of $g$
equals to the 1-th coefficient of $f_g(X)$. With the assumption that
\lstinline|(hg_unit : IsUnit (g.coeff 1))|, then
\lstinline|IsUnit ((RecurFun ht hq σ s hg).coeff 1)|. Therefore, there is a substitution inverse
of $f_g(X)$ which denote as $f_g^{-1}(X)$.

\begin{lstlisting}
lemma coeff_one_RecurFun: (RecurFun ht hq σ s hg).coeff 1 = g.coeff 1 := ...

lemma coeff_RecurFun_unit (hg_unit : IsUnit (g.coeff 1)) :
    IsUnit ((RecurFun ht hq σ s hg).coeff 1) := ...

def inv_RecurFun := PowerSeries.subst_inv _ (coeff_RecurFun_unit ht hq σ s hg hg_unit)
  (constantCoeff_RecurFun ..)
\end{lstlisting}

% Furthermore, let $F_g(X,Y) = f^{-1}_g(f_g{X} + f_g(Y))$ and $F_g(X,Y)$ is
% a commutative formal group law over $K$. And according to the functional equation lemma
% \ref{thm: fun_eq}, $F_g(X,Y)$ is a commutative formal group law over $R$.
\begin{lstlisting}
def inv_add_RecurFun :=
    (inv_RecurFun ht hq σ s hg hg_unit).subst
      ((RecurFun ht hq σ s hg).toMvPowerSeries (0 : Fin 2) +
        (RecurFun ht hq σ s hg).toMvPowerSeries 1)
\end{lstlisting}

The API \lstinline|inv_add_RecurFun| is the multivariate power series
$F_g(X,Y) = f_g^{-1}(f_g(X) + f_g(Y))$. And
\lstinline|(RecurFun ht hq σ s hg).toMvPowerSeries (0 : Fin 2)| represents the multivariate
power series $f_g(X)$, where $X$ is the first indeterminate variable in two variate power
series. Similarly, \lstinline|(RecurFun ht hq σ s hg).toMvPowerSeries 1| represents $f_g(Y)$.

Recall that the first part of functional equation lemma $\ref{thm: fun_eq}$ is that
$F_g(X,Y)$ has its coefficient in $R$. Moreover, this is a commutative formal group law due to
the definition of $F_g(X,Y)$. The first part of functional equation lemma is formalized as follow.

\begin{lstlisting}
theorem coeff_inv_add_mem_Subring [UniformSpace K] [T2Space K] [DiscreteUniformity K]
    (hs₀ : s 0 = 0) : ∀ n, (inv_add_RecurFun ht hq σ s hg hg_unit).coeff n ∈ R := ...
\end{lstlisting}

Next, we first prove that $F_g(X,Y)$ is a formal group law over $K$. The associativity condition
follows that
\[F_g(F_g(X,Y), Z) = f_g^{-1} (f_g(X) + f_g(Y) + f_g(Z))=  F_g(X, F_g(Y,Z))\]

Then we use the API \lstinline|FormalGroup.toSubring| and the theorem
\lstinline|coeff_inv_add_mem_Subring| to get a formal group law
\lstinline|CommFormalGroup.InvAdd_RecurFun| over $R$.

\begin{lstlisting}
def CommFormalGroup.InvAdd_RecurFun_Aux : CommFormalGroup K where
  toFun := inv_add_RecurFun ht hq σ s hg hg_unit
  ...

def CommFormalGroup.InvAdd_RecurFun [UniformSpace K] [T2Space K]
    [DiscreteUniformity K] (hs₀ : s 0 = 0) : CommFormalGroup R :=
  (CommFormalGroup.InvAdd_RecurFun_Aux ht hq σ s hg hg_unit).toSubring _
    (coeff_inv_add_mem_Subring ...)
\end{lstlisting}

Recall that the second part of the functional equaion lemma $\ref{thm: fun_eq}$ is that
for any power series $g, g'$ over $R$ with the assumption that the constant coefficients of $g,g'$
is zero, then $f_{g'}^{-1} (f_g(X))$. Then the second part of the functional equation lemma
is formalized as follow. In the lemma below, \lstinline|f_g'| is the power series $f_{g'}(X)$ and the \lstinline|G| is
$f_{g'}^{-1}(f_g(X))$.

\begin{lstlisting}
lemma coeff_inv_RecurFun_g'_mem_Subring [UniformSpace K] [T2Space K]
    [DiscreteUniformity K] (hs0 : s 0 = 0) :
    let f_g' := (RecurFun ht hq σ s hg')
    let G := (inv_RecurFun ht hq σ s hg hg_unit).subst f_g'
    ∀ n, PowerSeries.coeff n G ∈ R
\end{lstlisting}

The third part of the functional equation lemma is that there is a power series $\hat{h}$ over $R$,
such that $f_g(h(X)) = f_{\hat{h}}(X)$. According to the functional equation
$\ref{equation: recurFunEq}$, it is equivalent to show that the power series
$f_g(h(X)) - \sum_{n= 1}^{\infty} s_n \cdot \sigma^n f_g(h(X^{q^n}))$ is a power series over $R$.
And this is exactly the desired power series $\hat{h}$.

I would explain some details in part four of the functional equation lemma briefly.
To begin with, \lstinline|α.coeff n - β.coeff n| is a element in $K$, since $\alpha$ is
a power series over $R$ and $\beta$ is a power series over $K$. Then we need to formalize it
as a element in a set of $K$ rather than in ideal of R.
Secondly, \lstinline|R.subtype '' ↑(I ^ r)| is a set of elements in $K$ which is image of $I^r$
under the natural algebraic map $R \to K$. Moreover, the symbol \lstinline|↑| is necessary
since there is a coercion from \lstinline|Ideal R| to \lstinline|Set R|.
